# redsdice
Command to get dice rolls in DnD notation

```
Usage: redsdice <DnD dice descriptor> [...]

Examples:
redsdice d20
redsdice 4d18 d12 1d8 1d34+8

DnD dice descriptor:
[number of dice]d<number of sides per die>[+-modifier]
2d6+8 means throw two six-sided dice, sum them, then add 8

```

This was inspired by rolldice.
